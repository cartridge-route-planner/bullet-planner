🔑 GitHub 資料庫設定

設定說明：
1. 用戶名：你的 GitHub 帳號
2. Repository：例如 cartridge-planner
3. Token：GitHub → Settings → Developer settings → Fine-grained tokens，權限選 Contents: Read and Write
4. 資料目錄：預設 data（存放 routes.json 及 licences.json）
⚠️ Token 只存於本機 sessionStorage，關閉分頁後自動清除。
